# فروشگاه زعفران آنلاین

این یک سایت فروش آنلاین زعفران است که به صورت آفلاین و تک فایل HTML ساخته شده است.  
ویژگی‌ها:
- طراحی مدرن و تم پاستلی
- امکان افزودن و ویرایش محصولات از طریق پنل مدیریت
- سبد خرید با افزودن تعداد، حذف و مشاهده جمع کل
- وارد کردن شماره تماس و تکمیل خرید (شبیه‌سازی شده)
- بدون نیاز به سرور و قابل انتشار روی GitHub Pages

## انتشار
1. فایل `index.html` را در ریپازیتوری آپلود کنید.
2. در تنظیمات GitHub Pages شاخه `main` و فولدر `root` را انتخاب کنید.
3. لینک سایت در GitHub Pages فعال خواهد شد.
